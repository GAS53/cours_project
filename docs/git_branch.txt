Руководство как получить из гита репозиторий и создать свою ветку: 

Возможно все делать с графическим интерфейсом программой типа gitcracken или подобной
ниже представлены команды для терминала.
все команды выполняются в терминале(у меня linux тожесамое можно в windows cmd):

cd course_project  # заходим в папку которую сделаем рабочей папкой проекта 
git init # инициализируем гит (создаются его базовые файлы)
git remote add origin git@github.com:GAS53/cours_project.git  # привязываемя к моему гиту
git pull # получить репозиторий из сети
git checkout dev # перейти на ветку dev появиться папка app 
git checkout - b <свой ник> # создаем свою ветку и в ней уже можно делать все что угодно


очередность git checkout важна для правильного наследования веток
Если гит потребовал настройки:
    git config --global user.name "<Имя/ник>"
    git config --global user.email "электронная почта"
    git config --global color.ui auto
